OpenBroker Connector for RealHomes Theme. Populate your RealHomes objects with OpenBroker API.

== Description ==
* Author: [teamopenbroker](https://openbroker.com)
* Project URI: <https://openbroker.com>

This is official OpenBroker.com plugin which enables WordPress users to build Real Estate websites based on RealHomes theme.

= Tags =
openbroker, openbrocker, real estate, properties, realhomes, real homes

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly;
2. Activate the plugin through the 'Plugins' screen in WordPress

= How to use this plugin? =
1. Open your plugin settings page in WordPress Dashboard and enter your API credentials (can be found at your OpenBroker dashboard)
